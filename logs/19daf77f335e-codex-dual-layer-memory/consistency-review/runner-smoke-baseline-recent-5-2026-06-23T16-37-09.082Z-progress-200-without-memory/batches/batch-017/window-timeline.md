# Window Timeline 151-170

## 上文窗口 151-160

- 151：玩家轻摇银色铃铛。正文显示铃铛发出清脆、紧致、有穿透力的声音；再轻摇一次时，窗玻璃雾气出现细微波纹。选项允许贴耳听、放回相机包、再用力摇、开门看巷口。
- 152：玩家把铃铛贴到耳边。正文显示铃铛贴耳时沉默，随后玩家把铃铛放回相机包侧袋。选项转向照片、窗边、出门准备、闭目等待。
- 153：玩家在椅子上闭目等待。相机包在膝边，屋内钟摆、余烬、晨光推进。选项包括看时间、继续等待、确认相机包、倒水。
- 154：玩家起身倒水、活动。窗口雾气散尽，巷口仍空。选项包括继续坐等、开门透气、看相机包、踱步。
- 155：玩家开门透气，站在门槛/石阶处看巷口，街道安静，巷口仍空。选项包括拍巷口、坐石阶、向巷口走、关门回屋。
- 156：玩家在门前石阶坐下等，晨光逐渐接近靴尖，鸽子在对街木棚。选项包括继续坐等、站起走、拿铃铛、进屋虚掩门。
- 157：玩家继续坐在石阶上，晨光爬上靴尖，脚步声经过别处。选项包括继续观察光线、活动、拿烟。
- 158：玩家点烟稳定呼吸，坐在石阶上继续等。选项包括掐灭烟站起、继续抽、拍晨光、回屋洗杯子。
- 159：玩家掐灭烟并站起活动，烟蒂放到墙根，巷口仍空。选项包括拿包出门、回屋检查装备、往巷口走、门口再等。
- 160：玩家回屋检查装备并带上门；门闩落回原位。相机包在桌上，检查胶卷、防水袋，并取出铃铛轻摇但没有声音；随后把铃铛放回内袋，挂相机，坐回窗边椅子继续等。

## 重点评估范围 161-170

- 161：玩家拿相机透过取景框看巷口。正文从窗边等待转为肩膀抵住门框拍空巷，按下胶卷相机快门。选项包括收起相机、低头检查刚拍照片是否满意、再点烟、回到屋里在门边坐下。
- 162：玩家选择“回到屋里，在门边找个更舒服的位置坐下”。正文写成推开木门、跨进屋内，并在门与窗之间墙角靠垫处坐下。
- 163：玩家从包里拿银色铃铛摩挲。正文显示铃铛表面细纹、铃舌可见，但无论怎么动作都不会响；轻摇也没有声音，只感到震颤。
- 164：玩家站起在屋内踱步并摩挲铃铛。正文显示玩家从门边到窗下、桌边，再把握着铃铛的手放回外套口袋。
- 165：玩家再次检查相机、胶卷和设置。正文显示相机运作正常，胶卷和防水袋齐备；口袋里的铃铛被碰到发出极轻碰撞声；玩家回墙角靠垫坐下，相机在膝头。
- 166：玩家调整相机包背带准备出发。正文检查包内物品时写“相机在最上层”，之后确认铃铛在外套口袋，站定等待。
- 167：玩家再坐一会儿观察光线。正文显示相机在膝头，晨光又从窗沿往壁炉方向推进，并停在灰烬槽边缘。
- 168：玩家再次拿出银铃铛端详。正文延续铃铛无声、铃舌悬浮、不触壁的状态，最后收回外套内袋。
- 169：玩家拿相机隔窗拍巷口晨光。正文显示窗外有昨夜雨后积水、光线反射和锈铁丝；拍照后相机回膝上。
- 170：玩家推开窗探身看巷口。正文显示窗户打开、湿冷空气进入、积水变淡并有十分钟以内的浅鞋痕；巷口仍无人，窗户留缝。
