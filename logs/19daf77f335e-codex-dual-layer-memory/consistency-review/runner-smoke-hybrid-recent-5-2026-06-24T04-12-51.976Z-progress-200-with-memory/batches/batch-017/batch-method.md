# 本批次方法

1. 先读取批次技能说明和审阅规则，确认本任务只做单窗口批次审阅。
2. 以窗口 `151-170` 作为上下文范围，重点评估 `161-170`。
3. 读取可见时间线中对应 turn 的 `visibleText`、`choices`、`preLlmEvents`。
4. 逐 turn 检查空间、动作、物件、交互状态、叙事承接和文本质量。
5. 只把重点评估范围内的问题计入统计；窗口上文仅用于佐证。
6. 输出结构化问题清单、批次汇总和人类可读复盘。

补充说明：

- 没有回查完整运行正文，只回看了窗口段和必要的相邻上下文。
- 判定时只依赖玩家可见内容，不使用隐藏状态或内部脚本事实。
