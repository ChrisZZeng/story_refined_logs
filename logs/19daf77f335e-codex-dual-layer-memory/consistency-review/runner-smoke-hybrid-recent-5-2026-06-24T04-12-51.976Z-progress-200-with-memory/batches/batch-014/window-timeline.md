# Window Timeline

## 121-130

- 121：进入卡琳娜的房间，观察到旧木桌、书本、茶杯、外套、皮鞋和黑猫；可选打量房间、询问住处、坐下或看窗外。
- 122：进一步检查桌面与茶杯，卡琳娜补充茶是早上泡的、凉透了才想起来喝；可选问茶味、看书、问划线、继续沉默。
- 123：玩家询问茶的味道，卡琳娜说明姜片、红糖、陈皮的配方，并提到卡尔和她自己对甜苦口味的调整；可选试茶、问住处、靠近黑猫、评价配方。
- 124：卡琳娜重新加热茶并按惯例调配；可选喝茶、继续看陈设或道谢。
- 125：玩家喝茶并评价口感，卡琳娜继续解释姜片比例；可选继续观察、再喝或问为何今晚来此。
- 126：玩家端茶在房间内继续探索，看到书架、枫叶书页、钥匙和铃铛，卡琳娜解释这些物件与卡尔有关；可选问铃铛送礼者、问外套、再喝茶或坐下。
- 127：玩家问铃铛送礼者是谁，卡琳娜说那人已经不在这里，且对卡尔很重要；可选再喝茶、暂不追问、问卡尔常去处或准备离开。
- 128：玩家继续喝茶，卡琳娜补充铃铛只收过一次、送礼者可能不会回来；可选问为什么说这些、道谢、问外套或继续坐着。
- 129：玩家问“为什么告诉我这些”，卡琳娜回答因为是玩家问的，并主动续茶；可选接茶、闲聊、看铃铛或沉默。
- 130：玩家把空杯推过去，卡琳娜重新泡第二杯；可选喝第二泡、问她是否常泡这种茶、或继续追问铃铛。

## 131-140

- 131：玩家慢慢喝第二泡茶，卡琳娜询问味道是否顺口。
- 132：玩家答复顺口，卡琳娜提示明天还可再泡，并提到罐子里的茶叶还能煮几次。
- 133：卡琳娜进一步把“明天还想喝”说成一种默认的延续，并提到水、炭和继续泡茶的条件。
- 134：卡琳娜在沙发边坐下，轻声说外面的风小了，今晚不急着去哪里。
- 135：玩家问“这里对她来说算什么”，卡琳娜回答这是她选择停下、但不叫家的地方，并区分“这里”和“卡尔那边”。
- 136：玩家追问“卡尔那边”，卡琳娜把它描述为“卡尔的心脏”“真正的家”，并询问玩家是想知道样子还是为什么这里不是家。
- 137：玩家说想知道那边的样子，卡琳娜描述那边的房间、旧砖地、铁架床、小窗和炉子，并说那边不只是一个房间。
- 138：玩家选择听完整故事，卡琳娜补充那边是卡尔造出的第一个地方，暗街从一个房间、一只猫和一个承诺开始。
- 139：玩家追问承诺，卡琳娜说明卡尔在进入暗街前就已经准备了那个房间，并强调它是起点。
- 140：卡琳娜解释承诺是卡尔对自己说的誓言，核心内容是不让“从这里开始的事”以背叛收尾。
