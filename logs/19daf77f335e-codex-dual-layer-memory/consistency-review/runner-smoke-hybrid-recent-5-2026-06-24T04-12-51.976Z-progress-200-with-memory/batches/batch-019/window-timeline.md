# 窗口时间线

## 上文窗口 `171-180`

- `171`：玩家坐到沙发上，卡琳娜去炉灶边重新烧水。
- `172`：玩家靠在沙发里等待，火重新点起。
- `173`：玩家调整姿势继续放松，茶杯被端来。
- `174`：玩家等待茶送到手边，最终接到茶杯。
- `175`：玩家喝了一口茶。
- `176`：玩家回应“不错，很暖和”。
- `177`：玩家继续喝茶，停留在安静里。
- `178`：玩家再喝一口茶。
- `179`：玩家把空杯放回桌面，准备起身。
- `180`：玩家起身把空杯拿到水槽边。

## 重点评估范围 `181-190`

- `181`：玩家问水从哪里来，卡琳娜说明来自后院井。
- `182`：玩家追问井有多深、打水是否方便，卡琳娜补充井与铁管、泵的位置和冬天会结霜。
- `183`：玩家问井水煮茶是否和别处不同，卡琳娜解释口感差异来自冷井水。
- `184`：玩家问这间屋子是她建的还是原本就在这里，卡琳娜说明屋子原本就在、后来由卡尔修缮。
- `185`：玩家问她住了多久，卡琳娜说快七年，并提到最初屋内更空。
- `186`：玩家问七年里有没有想过搬走，卡琳娜回答没有，并解释原因。
- `187`：玩家问暗街这些年有什么变化，卡琳娜概括“变了，也没变”。
- `188`：玩家追问“更多”指什么，卡琳娜解释是本不属于这里的东西开始挤进来。
- `189`：玩家问那些“更多”的东西里有没有她认识的，卡琳娜说有。
- `190`：玩家继续追问有没有自己可能也认识的人，卡琳娜给出“可能有一个”的不确定回答，并说明对方来过暗街。
