- 保持神秘感本身就是叙事的一部分

你的手指已经搭上了相机的快门，随时准备举起。沙发靠背上，那只黑猫的尾巴尖轻轻一甩——它也没有动，但姿态已经从慵懒切换成了一种微妙的警觉。
门外站着一个人。
你只能看到一部分——一只手插在口袋里，半张脸被楼道里昏暗的光线遮住。是个男人，穿着深色的外套，肩膀处有水渍的痕迹——外面开始下雨了。
“卡琳娜阁下。”
他的声音压得很低，带着一种刻意的恭敬。他微微低下头，没有往里看——至少没有明显地在往门内张望。
“有件事要跟您说一下。费舍尔那边的人，今天在港区翻到了一条船。船舱里没人，但有货——跟您之前嘱咐我们留意的东西吻合。”
他停顿了一下。
“还有一件事。凯旋门的人今天下午在暗街入口拦了一个外地人——他们已经把这事报到上面去了。您可能需要知道。”
卡琳娜没有立刻回答。她能感觉到你的目光交织，但她始终没有回头。
“凯旋门的事我已经知道了。”她的声音平稳，带着那种你之前见过的、让黑帮退避三舍的冷淡，“费舍尔的货——你说‘吻合’，具体是什么？。”
“暂时说不准。负责盯梢的人没敢靠太近，只记下了编号。他们说看起来像是船运箱子，密封的，贴着老标签——可能是军火，也可能是其他东西。”
卡琳娜的手在门把上轻轻叩了一下。
“我知道了。你回去跟费舍尔说，那批货先别动。我不在场的时候，谁都不许掀盖子。”
“明白。”
男人点了点头，转身往楼梯口走去。脚步很轻，很快就消失在楼道的转角处。
卡琳娜关上门，锁舌再一次咔哒入位。她靠在门板上，微微仰了仰头——那一瞬间，她脸上的冷淡消退了一些，像是短暂地卸下了什么。但只过了一两秒，她就恢复了正常。
“看来今天热闹的事情不止你这一件。”
她走回沙发边，重新坐下。黑猫睁开一只眼看了她一下，又阖上了。
她看向你，目光里那层疏离已经退了，取而代之的是一种略带疲惫的平静。
“我刚才说到哪了？——对，你讲她的事。讲完我会告诉你她离开前说的那句话。”
她靠进沙发靠背，双手交握放在膝上，等着你开口。
窗外，雨声渐渐大了起来。
