# Window Timeline

## Turns 1-10 Context

1. 帕兹从关于敏特死亡的血色噩梦中醒来，身处新西西里复活日庆典。金发蓝眼少女卡琳娜拿着他的记者证，称他为酒鬼并离开。小贩介绍复活日、十年前势力洗牌和新西西里的黑帮管事现状。
2. 帕兹询问卡琳娜，小贩称她是暗街的调停人，收取的是“亏欠”而非金钱。
3. 帕兹离开摊位进入庆典人群，寻找红发踪迹。
4. 帕兹回到小贩摊位打听庆典，小贩提醒他在特殊日子里到处打听会显眼。
5. 帕兹观察普通人的复活日，看到父女、手风琴老人、醉汉和爆竹，战地创伤反应反复出现。
6. 帕兹追问为什么打听会显眼，小贩解释复活日是大家假装忘记的日子，管事的人不喜欢外来者打探。
7. 帕兹指出小贩不像普通小贩，小贩承认自己在街上摆摊八年、知道暗街与卡琳娜，并提醒他记者证不值钱。
8. 帕兹说自己在找一缕红发，小贩回忆两天前似乎在暗街附近见过红发女子。
9. 小贩进一步说明两天前傍晚在古董街、老钟楼对面旧物铺附近的巷口看到红发女子。
10. 帕兹前往古董街，在钟楼、旧物铺和巷口附近确认小贩描述的位置。

## Turns 11-20 Evaluated

11. 帕兹绕钟楼寻找观察角度，发现巷口墙根凹陷里有几根铜红色发丝，判断敏特曾在这里站过。
12. 帕兹把发丝夹进敏特照片，沿暗街方向深入，发现半个鞋印、碎布、分岔路和右侧的焦纸味。
13. 帕兹检查鞋印和碎布，推断脚印约一天半到两天，碎布是自然撕裂，线索指向左边窄巷；同时注意暗处有影子动过。
14. 帕兹站直并对暗处发问“谁在那儿”，暗处的影子停住。
15. 帕兹用相机闪光灯照亮暗处，看见金发、白色棉衣、吊带裙、蓝眼睛的人影。
16. 帕兹保持警戒观察，确认暗处的人是庆典上见过的卡琳娜；她问他在暗街做什么。
17. 帕兹保持相机姿势问卡琳娜是否认识敏特；卡琳娜重复“敏特”，并问他是欠债还是讨债。
18. 帕兹放下相机，坦白自己不是讨债而是在找一个重要的人；卡琳娜重复他的话并重新打量他。
19. 帕兹直接说明在找敏特并问卡琳娜是否见过；卡琳娜确认他是庆典上到处打听的酒鬼记者，并尚未开口回答。
20. 帕兹问卡琳娜到底是什么人；卡琳娜回答自己是暗街的调停人，并承认认识敏特、见过她，引导帕兹顺着焦纸味继续寻找。
