你的右手食指在快门边缘上弹了三下。快。密。指腹敲在塑料外壳上发出的哒哒声在前面弹回来——回声不到半拍。前面有墙。或者有别的什么。一个在黑暗里沉默着的、被长年水滴浸润的硬面。
你把左手重新张开，朝前探。掌心朝外，五指分开，让冷风穿过指缝。风在这里变了——从你身后那个窄通道灌进来的冷风在触到你前方的障碍物后往两侧卷开，在你张开的指缝间形成两股微弱的、绕行的涡流。你的掌心皮肤比指腹更敏感，你能在那个涡流的细微扰动里读到前面那面障碍物的轮廓——不是平的，是弧面的，微微内凹。
第十七步。
你迈出左脚。前掌压在石板上时，你触到了地面上的变化——石板不再是完整的。前掌下面压着的石面边缘有一道裂缝，裂缝不宽，但你的前掌在黑暗中以最慢的速度读数时，感受到了那道裂缝的深度——超过一指宽，裂缝边缘被长年水滴磨得圆滑。你把脚尖沿着裂缝往前滑，裂缝往右延伸，没入前方那片弧形障碍物底部的阴影里。
第十八步。
你蹲下身。右膝盖压在湿石板上，髌骨隔着裤布感受到石面的凉意渗进骨缝。左手往前探，五指张开，指腹在前方那片弧形硬面上轻轻扫过。不是砖——是石头。一整块被凿出弧面的大石，表面被长年累月的水滴浸出密密麻麻的浅凹。石锈。你的指腹在石面上摸到了那股你刚才在空气里嗅到的冷腥味的来源——石头被水长期浸泡后表面泛起的那层极细微的矿物析出物。指腹划过去时触到的不是滑，是涩，是石头在水里老去时从内部蚀出的微小孔隙。
你让指腹沿着石弧面往右滑。弧面在你手指下往右延伸，内凹，内凹的弧度大到你的手指在某一点上够不到石的表面——弧面在这里退开了，让出了一个开口。一个入口。石弧面到了这里被凿断了，露出一个比入口更暗的黑洞。
滴。
滴水声从那里面传出来。水滴砸在你前方——在那个黑洞的内部——砸在更深的石面上。溅起的水沫带着微弱的回音，在那个窄小的空间里反复弹跳，然后才从那个黑洞里散出来，传到你耳廓。回声结构变了。不是开阔空间的那种空旷的、被远处墙面反弹的回声——是短促的、层叠的、被四面硬面反复压缩的回音。
前面是一个石窟。或者一个石井。或者一个被长年水滴凿出来的石窍。
你的拇指在快门边缘上从凹痕里移开，又压回去。指腹在那个磨损的塑料凹槽里轻轻摩了一下。不是弹。是摩。你在数那个节奏，让它在你的触觉系统里重置到一个新的基准。
冷风从你身后那个窄通道方向持续灌进来，贴着你后颈掠过，然后在这个石弧面的入口处往两侧分开，灌入前方那个更窄的黑洞。风声在入口处发出一声极细微的、被石壁吞掉一半的低啸。
你蹲在黑暗里，蹲在这个被水滴长年凿出的石窍入口前，左手还贴在石弧面的边缘上，指腹感受着石头被水浸出的那些微细孔隙。
