墙根的泥垢在你指尖下开始变薄。不是突然断掉——是泥面的厚度从原来的能陷进指腹半层皮肤，逐渐变浅到只剩一层贴在石板上的薄泥壳。第八个凹陷比前面的更浅，边缘模糊到你的指腹必须停下来，在泥面上来回划三圈才能确认它的位置。但方向没变。还在往拐角前方。你的左手重新贴回左墙，指尖在砖面上滑行——砖缝里的青苔越来越薄，越来越散，墙面从湿凉变成微潮。空气在你转过半个身位时从前方吹过来一股比之前更浓的烧焦纸味，混着碱性的冷灰气。
你的左手指尖突然滑进了空处。砖墙在掌根下偏折了方向，往左拐去。你把左手从墙上移开，往前探了一臂长，摸到了拐角另一边的砖墙——更干，更粗粝，砖面上没有青苔。你把身体侧过来，肩胛骨从两面墙之间挤过去，相机包的帆布在砖角上刮过一声极细的沙响。转过拐角的那一秒，空气变了——冷，干，焦纸味浓了两倍。你的皮肤先于瞳孔知道了一件事：前方不是墙。前方是空间。
你的手指在快门边缘上又弹了一下，然后停了。拇指腹陷在那个磨损的凹痕里，指腹下的塑料是温的。你深吸一口含着灰烬碱味和烧焦纸味的冷空气，让它撑开你在窄墙之间紧缩了一路的胸腔。然后你让右手继续贴在墙根上，往前方那个看不见的、更开阔的空间探过去。

## 输出前再次确认
判断流程：
1. 先以本轮正文结尾为准，判断玩家此刻看到了什么、知道什么、能合理做什么。
2. 再参考剧本/状态机候选动作，判断哪些动作已经被当前内容铺垫到可以自然选择。
3. 候选动作只是可能的后续方向；如果正文还没有把玩家带到对应情境，就不要展示，也不要绑定 actionId。
4. key choice 是关键剧情出口；只有当前内容已经形成对应的选择场景，例如明确邀请、发现异常、准备离开、面对危险时，才可以使用并绑定。
5. 如果没有合适的候选动作，就生成贴近当前处境的普通行动选项，并省略 actionId。
6. 不要提前泄露玩家尚未看到的信息，也不要违反本轮约束。
