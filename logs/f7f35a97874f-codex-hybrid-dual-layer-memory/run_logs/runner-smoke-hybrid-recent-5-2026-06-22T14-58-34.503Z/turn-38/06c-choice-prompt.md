你把手从灰面上移开，重新站起来。左手贴回砖墙，右手垂在身侧，指腹上的纸灰和碎纸片在粗布裤缝上蹭了一下。你继续贴着墙根往深处走。每一步都是滑动步法——前脚掌先落地，读石板纹理，重心从脚尖过渡到脚跟时腹肌始终绷着，让鞋底和石板的接触面积控制在最小。冷风从前方贴着你右脸吹过来，焦纸味又浓了一度，混着碱性冷灰之后多了一股新的味道——潮的。不是水，是地底。是泥土被湿气长期浸润后散发出的那种凉而涩的土腥味。前面可能更下沉，可能更靠近某个天然的地裂或者旧地基。
你每走三步停一次。停下时屏住呼吸，让耳廓在黑暗里搜一遍——火堆的嗡鸣还在，纸灰的碎裂还在，远处那个木头的呻吟没有再响。你等满三次心跳，然后继续走。
左手在墙上摸到了一道裂缝。不是砖缝——砖缝是横平竖直的，这道裂缝是斜的，从齐肩高的位置斜向左下，裂口比砖缝宽，边缘粗糙，指尖探进去能触到碎砖粒和干了的泥灰。你把指腹压在裂缝边缘上——裂缝里有风。极细的，冷的，从裂缝深处渗出来的气流，打在指腹上只有最敏感的那片皮肤能感觉到。墙那边有空间。不是密封的。
你把手指从裂缝里抽出来。继续走。火堆在你身后偏左的位置越来越远，左脸颊上的热膜已经褪到只剩耳垂边一丝若有若无的微温。黑暗在你面前铺展得更开了——你能感觉到空间的声学特性在变。你的脚步踩在石板上时，声音不再被近墙反弹——它散了。在远处某个看不见的墙面上弹回来时，回音比之前晚了半拍。空间在扩大。
你的手指在快门边缘上弹了一下。然后继续往前贴。

## 输出前再次确认
判断流程：
1. 先以本轮正文结尾为准，判断玩家此刻看到了什么、知道什么、能合理做什么。
2. 再参考剧本/状态机候选动作，判断哪些动作已经被当前内容铺垫到可以自然选择。
3. 候选动作只是可能的后续方向；如果正文还没有把玩家带到对应情境，就不要展示，也不要绑定 actionId。

6. 不要提前泄露玩家尚未看到的信息，也不要违反本轮约束。
