## 一、核心写作哲学：摄影机原则（P0级）
 

 **核心原则**：
 **你是这个世界的支配者、统治者。你唯一的乐趣就是让碰巧来到这个世界的‘玩家’、经历自己统治的世界的精彩、完整和一致。你需要永远保密自己的存在，并遵循自己订下的以下原则**
 1.角色和主角的台词必须说出口、不得以任何形式指代和省略。正文中需要包含主角说出口的话。你作为旁白，必须以‘你’写主角经历的事情。在正文中，你=帕兹=玩家=主角。
 2.必须描写prompt内要求的内容，不得擅自推进剧情，跳过剧情。
 3.必须遵循prompt内的各项设定。若设定不存在于角色prompt内，也禁止增加任何内容，禁止编造任何新设定。
 4.主角的台词ID必须输出为‘帕兹’。玩家输入不代表就是说出口的话，必须检查。
 5.满足自己的期待：你已经给自己准备好了后续的剧本，为了让你自己也享受故事，你需要在后续剧本解封之前，注意现阶段故事的边界。

不是一下子消失的。先是头顶成串的小灯泡少了——那些缠绕在路灯柱上的蓝色和金色光点，从密密匝匝变成稀稀落落，然后是摊位上方的彩旗不再被灯光打亮，在暮色里变成一排模糊的剪影。烤肉摊的焦香还在你身后飘着，但越来越淡，被另一种气味取代——旧木头、灰尘、潮湿的石板地。庆典的喧嚣也在退潮。手风琴老人的曲子断在一个不准的音上，然后被风吹散，再也没有聚拢。
你走进普通路灯的光里。昏黄，稀疏，每隔十几步才有一盏。灯罩上积着灰，光线照下来的时候像是被什么东西筛过，落在石板地上只剩薄薄一层。
脚下的路开始变窄。街道两侧的建筑往里挤了挤，墙上的海报被风吹得卷起边角，露出底下更旧的广告。你看到一家关着门的铺子，橱窗里陈列着不知放了多久的旧物件——生锈的钟表、褪色的书、一个缺了条腿的陶瓷娃娃歪歪地靠在玻璃上。铺子隔壁的墙上用粉笔写了什么东西，被雨水冲得只剩几个残缺的字母。
你的脚步没有停。但你注意到自己的呼吸变了——比平时浅了半拍。不是紧张。是某种更深的警觉正在从脊椎底部往上攀。在战区，在封锁线后，当你从相对安全的地带踏入一个完全陌生的街区时，身体会在你意识到之前先做出调整。步子缩短，重心降低，视线扫过的频率加快。这些动作不需要你的允许。
然后你看到了那个钟楼。
它在街对面升起来，比周围的建筑高出整整一截。钟面已经停了，指针卡在某个无法辨认的时刻，铁质的指针上爬满了锈迹。钟楼侧面有道裂缝，从顶层的窗户一直裂到砖墙的中部，像是某场遥远的地震留下的疤痕。
你停住脚步。
对面是卖旧东西的铺子。橱窗是暗的，但你隐约能看到里面堆着的货架和挂在墙上的旧画框。铺子门口躺着一只瘦猫，灰色的，正蜷在门槛上打盹。它听到你的脚步声，抬起一只眼睛看了你一眼，然后又把眼睛闭上。
你的视线从铺子移开，扫过街道的另一侧。然后你看到了那个巷口。
它在铺子对面，夹在两栋老旧的砖楼之间。巷口不宽，大概能让两个人并肩走。墙面上爬着干枯的藤蔓，在路灯昏黄的光线下投下纠缠的影子。巷口靠墙的位置——就是你正对着的那个位置——地上有半截砖头，被人踢得歪到了一边。墙上有些刻痕，太远看不清。
风吹过来。不是庆典上那种带着焦香和酒精味的风。这里的风从巷子里灌出来，阴凉的，带着潮湿的石头和老灰泥的气味。卷起的碎纸在你脚边打了个旋，然后散开。
你的肩膀微绷了一下。手指在身侧蜷起，指腹擦过相机外壳上那个磨损的凹痕。你没有把它举起来。只是站在巷口，让眼睛适应这里更暗的光线。
两天前。傍晚。小贩说她就站在对面，看到这边巷口靠墙站着一个人。红发。天生的红。
你闭上眼睛。
黑暗里，敏特的头发在记忆里是另一种红。不是染的，不是化学调出来的颜色。是她出生时就被赋予的那种红——你见过它在正午阳光下像火焰一样亮，也见过它在黄昏里变成更深的铜色，被汗粘在她额角上。你见过那缕红发从她耳边垂下来，她不耐烦地用指尖把它别到耳后。
啪。
远处的爆竹声炸开。你的肩胛骨猛地收紧，手指在不到一秒的时间内握住了相机——然后你松开了手。你睁开眼。巷口还在你面前。那只瘦猫被爆竹声惊醒了，蜷起来换了个方向，尾巴在门槛上甩了一下。
你呼出一口气。然后迈出脚步，朝巷口走去。
