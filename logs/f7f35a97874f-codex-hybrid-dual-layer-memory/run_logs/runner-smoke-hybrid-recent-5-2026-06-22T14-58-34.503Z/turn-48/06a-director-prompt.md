## 一、核心写作哲学：摄影机原则（P0级）
 

 **核心原则**：
 **你是这个世界的支配者、统治者。你唯一的乐趣就是让碰巧来到这个世界的‘玩家’、经历自己统治的世界的精彩、完整和一致。你需要永远保密自己的存在，并遵循自己订下的以下原则**
 1.角色和主角的台词必须说出口、不得以任何形式指代和省略。正文中需要包含主角说出口的话。你作为旁白，必须以‘你’写主角经历的事情。在正文中，你=帕兹=玩家=主角。
 2.必须描写prompt内要求的内容，不得擅自推进剧情，跳过剧情。
 3.必须遵循prompt内的各项设定。若设定不存在于角色prompt内，也禁止增加任何内容，禁止编造任何新设定。
 4.主角的台词ID必须输出为‘帕兹’。玩家输入不代表就是说出口的话，必须检查。

第一层声音是那个石窍内部的水滴。滴。水珠从某个看不见的高处坠下来，砸在更深的石面上——不是你在铁钟下听过的那种沉闷的、打在金属上的撞击。这一次的滴水更脆，更短，水珠在石面上炸开的瞬间被四面硬壁反复弹回来，在你耳廓里层叠成一个压缩过的、短促的回音簇。主声之后是次声——水沫溅碎，洒在周围石壁上，每一滴碎珠打在弧面石壁上又弹出更细微的回音，在那个窄小的空间里反复折返，最后才从入口处散出来，传到你耳廓时已经被压缩成一道极细的、层叠的余响。
那里面是一个被水凿出来的石窍。不大——回音太短，太密，空间不会超过几米见方。但很深。水滴从坠落点到石面的距离，你从主声和次声之间的间隔判断，至少有五六米。
你的手指在快门边缘上弹了三下。快。密。指腹敲出的哒哒声在你脚边的湿石板上散出去，被前方那个石弧面弹回来——回声不到半拍。石壁比你想象的更近。入口不是敞开的——是收拢的，石弧面在入口处往内收了一道窄口。
第二层是风。冷风从你身后那个窄通道方向持续灌进来，贴着你后颈掠过，然后在石弧面的入口处被打乱了。气流撞上石壁后往两侧卷开，一部分从你张开的左手指缝间穿过——凉的，涩的，带着石锈的冷腥味——另一部分灌进了那个石窍入口。灌进去的风在窄口处加速，发出一声极细微的、被石壁吞掉一半的低啸。那声低啸传进石窍内部，在里面被四面硬壁反弹，变成一道几乎听不清的、被空间压缩过的闷响。
你的右眼皮在气流里细微地颤了一下。
第三层是火堆。在你身后，现在已经退到很远。嗡鸣不再是那种闷钝的低频气流，而是被开阔空间的空旷和前方石壁的反射搅成一道若有若无的、像是远处某台老机器在空转的底噪。纸灰还在碎裂。但那声音也远了——细碎的噼啪声传到你耳廓时只剩一片极细微的、被空间磨损的脆响余烬。
第四层是远处那根木头。在火堆嗡鸣的间隙里，在滴水声第三次滴落之后——那声低沉的、被自身重量压出的极慢呻吟传到你耳廓时只剩一丝几乎分辨不出的余震。你等了六次心跳。木头没再响。
你的拇指从快门凹痕里移开，又压回去。指腹在那个磨损的塑料凹槽里轻轻摩了一下——不是你刻意的动作。是你拇指肌肉在持续紧张状态下的一次细微抽搐。你让它抽。你数着那个节奏，让它在你的触觉系统里重置到一个新的基准。
然后你把注意力重新拉回石窍内部。
水滴的节奏是规律的——但间隔很长。每一声滴落之间的静默长到你能数完六次心跳。你在心里把那个节拍锁定：六次心跳，一滴。水珠砸在石面上的主声每次都有极细微的差异——不是音高，是溅碎后的次声分布。有时候水沫洒在左侧石壁上，有时候洒在右侧，有时候溅得更高，打在石窍顶部的弧面上，弹回来的回音多了一层。
水滴的落点可能在石窍底部一个被长年凿出的深石窝里。不是你在铁钟下摸到的浅凹——是更深的，积水更多，水珠砸进去时溅起的水沫分布范围更大。
你的左手从石弧面上移开，五指张开，掌心朝前——不是挡，是接。你在等下一滴水滴落，等那些溅起的水沫从入口处飘出来，打在你的掌心上。你等了六次心跳。滴。水珠在石窍内部炸开，细碎的水沫有一部分从窄口处飞出来，洒在你掌心上——凉的，细的，还没散开，在掌心皮肤上张成几个微小的半球。
你的拇指在快门边缘上弹了一下。
石窍内部没有其他声音。没有脚步。没有衣料摩擦。没有呼吸。没有任何属于人类的、有意图的移动。只有水在滴，风在灌，石壁在沉默地承受着长年累月的水蚀。
你把掌心收回来，五指并拢，让那些水沫沿着指缝滑进指尖。水是冰的——不是地底渗水的常温，是更冷的，冷到你的指尖在触到水沫的瞬间微微发麻。
你蹲在黑暗里，蹲在这个被水滴长年凿出的石窍入口前，左手重新贴回石弧面边缘，指腹感受着那些被水浸出的微细孔隙。右手的拇指还停在快门凹痕上，不弹了——只是压着，让那个磨损的塑料弧面抵住指腹的纹路。
你继续听。
