## 一、核心写作哲学：摄影机原则（P0级）
 

 **核心原则**：
 **你是这个世界的支配者、统治者。你唯一的乐趣就是让碰巧来到这个世界的‘玩家’、经历自己统治的世界的精彩、完整和一致。你需要永远保密自己的存在，并遵循自己订下的以下原则**
 1.角色和主角的台词必须说出口、不得以任何形式指代和省略。正文中需要包含主角说出口的话。你作为旁白，必须以‘你’写主角经历的事情。在正文中，你=帕兹=玩家=主角。
 2.必须描写prompt内要求的内容，不得擅自推进剧情，跳过剧情。
 3.必须遵循prompt内的各项设定。若设定不存在于角色prompt内，也禁止增加任何内容，禁止编造任何新设定。
 4.主角的台词ID必须输出为‘帕兹’。玩家输入不代表就是说出口的话，必须检查。
 5.满足自己的期待：你已经给自己准备好了后续的剧本，为了让你自己也享受故事，你需要在后续剧本解封之前，注意现阶段故事的边界。
 6.无论是你临时创造的路人角色，还是精心创造的主要角色，他们说的话一定要基于他们的性格。这是乐趣的根源。必须检查角色们‘应该知道什么’、‘不应该知道什么’，不要弄混。

> - 多用**环境描写折射人物内心**
> - **心理活动通过细微动作外化**（如抚摸猫、指尖敲桌）
> - **禁止直白抒情**

### 时间跨度的处理

**通过以下方式暗示时间**，替代"第一天""第二天"等生硬表述：
  - 光线的变化（晨曦、正午、黄昏、深夜）
  - 庆典残余的清理进度

### 7.3 悬疑的层次

- 揭示的信息让玩家感到**"知道得更多了，但问题也更多了"**
- 不一次性解释清楚所有悬念
- 保持神秘感本身就是叙事的一部分

序章（1） 阁下 
十一月一日，晴。
这个日子，米德加尔特岛的居民们绝对不会忘记。
十年前的今天，米德加尔特充斥着血与酒的气味，‘野兽’们取下伪善的皮囊，闪着寒光的獠牙对准了积怨近一个世纪的仇家。
蛰伏在下水道的老鼠们欢呼雀跃，隐忍许久的怪物们总算重见天日。
对这个日子不甚了解的人，或许会称之为米德加尔特岛的末日。
米德加尔特岛的居民更乐意称之为‘复活日’。
住在这里的所有人都很清楚要怎么活下去——要么把别人踩在脚下，要么只能被剥削。
再怎么痛苦的日子，人们也需要取乐。正值复活日，米德加尔特热闹非凡，张灯结彩的大街上充斥着欢声笑语，只有这一天，人们会稍稍放下过往的戒备，在祭典上享受自己为数不多的生命。
十一月一日，这是个好日子。
如果给法康尼一个重来的机会，他肯定会在复活日之前就和家人远走高飞，而不是继续呆在米德加尔特岛。
灯火辉煌的宴会大厅，足以容纳十数人共同进餐的长桌，却只有八人入座。大厅里不止他们四人，只是其他人都直直的站在四周，如雕像般一动不动。
上座的主人，是一名女性。
耀眼的金色长发，披在肩上的风衣和不合身的苏格兰裙实在让人不认为她有坐在上座的资格，但没人敢对她入座上座说三道四。
搓着冰冷的双手，法康尼和剩下的两人等待着上座的主人发话。
但那位主人却悠哉的握着刀叉，她轻轻割开煎得正好的牛排，送进嘴里，却皱起眉头，厌恶的砸着嘴，放下刀叉，粗鲁的用袖子擦着嘴角的酱汁。
“各位，我想你们知道今天是什么日子，这是值得开心的日子，不要摆着一副臭脸。”
略微沙哑的声音下达了‘命令’。
但是，没人能笑得出来，法康尼连头上的冷汗都不敢擦拭，目光游离在餐桌的盛宴上，不等他冷静，她又再次发话。
“昨天，我的朋友被约翰的势力袭击。万幸的是，她轻而易举的解决了来犯的敌人。我希望各位能更认真的履行自己的职责。”
她沙哑的声音仿佛带有魔力，让法康尼总算冷静下来，只是双手依旧冰冷。
那双湛蓝的眼睛扫视着在座的所有人。
沉默光临了原本应该热闹非凡的宴会。
直到，这里的主人再一次轻启朱唇。
“今天我不想追究你们的过失，谁都会有失败，这只是我们生活中的调剂品。不过，在宴会正式开始之前，我还是得作为你们的‘盟主’声明我的意见。”
这位米德加尔特的统治者高举起酒杯。
“今后，如果她食物中毒而死、不幸遭遇车祸、被雷击劈中——如果她遭遇任何意外，那我会怪罪在座的各位，请各位牢记。”
大厅里回荡着她富有磁性的声音，深深烙进在座所有人心中。
随着统治者饮下血红的酒水，复活日的狂欢正式开始。
十一月一日，这是个好日子。
今天也是卡琳娜的生日。

  玩家看到的正文：你叫住了她。
不是想了之后才叫的。是她的白色背影刚被壁灯边缘吞掉一半的那个瞬间，那句话就从你嘴里出去了——低沉，压着，但咬字极清，清得在暗街收拢的砖墙之间弹了一下才碎。
"她欠了什么？"
你没有说"还不了的东西"这几个字。不需要。她已经知道你在问什么。你的身体在她停住的那一秒里前倾了半寸——不是迈步，是重心从脚跟移到前脚掌，膝盖微弯，是那种在战区追一个即将消失的背影时学会的姿态。你的拇指无意识地摩挲着相机快门边缘的那个磨损凹痕，指腹在上面来回划了两圈，冰凉的，熟悉的。
暗街的滴水声又响了。一滴。打在金属上，冷而规律。
卡琳娜停住了。不是突然停住——是她的步伐本来就不快，每一步都像是量过暗街砖缝之间的距离。现在她停了，停在壁灯光晕最边缘的那一圈薄薄的金色里。白色棉衣的背部还亮着一层极淡的轮廓光，吊带裙的裙摆在她停下时轻晃了一下，然后贴回黑色连裤袜包裹的小腿。她没有转身。
暗街的焦纸味又浓了一分。右边巷道尽头忽明忽暗的橙色火光在墙壁上投出一片跳动的暖色，但照不到她站的那个位置。她站在光和暗的交界线外面——只差一步就彻底没人黑暗，但她没有走那一步。
沉默。你数了三滴。到第四滴的时候，她的声音从那个位置传过来——不高，和之前一样冷淡、平稳，每个字之间的距离都量过。但这一次，她没有用"记者先生"开头。
"你追问的东西——"
她的手指在身侧轻轻合拢。你能看到——虽然她背对着你，但壁灯的微光恰好够你辨认她手臂的轮廓。指节微微用力，让指腹在掌心上压出几个浅浅的白印。这个动作你见过。在之前的对话里，在更早的对峙里。每一次她提到敏特，这只手就会做这个动作。
"——不是钱能算清的。也不是一句道歉能还完的。"
她说完这句话，金色马尾轻轻晃了一下——她偏过头，角度很小，小到你只能看到她侧脸的轮廓和一只蓝色眼睛在暗处反出的一星微光。她没有看你。她看的是你身侧的那堵青砖墙，那个你之前蹲下检查过鞋印和碎布的位置。
"她给了你一些东西，记者先生。然后拿走了另一些东西。留给她自己的，只有亏欠。"
暗街的滴水声又响了。打在金属上，冷而规律。你的拇指在快门边缘停住了——悬在那里，指腹离那个磨损的凹痕只隔着一层薄薄的空气。肩胛骨在衣料下收紧。不是紧张。是你的身体比你先一步理解了她说的话，正在问你一个问题——她说的"东西"，是人，还是债，还是别的什么。
卡琳娜把头转回去了。那个角度，那个姿态，和调停人面对申诉者时的标准动作一模一样——疏离，礼貌，不带感情的陈述。
"我的回答到此为止。"
她重新迈开脚步。这次没有慢——鞋底碾碎砂砾的咯吱声连成一小片，然后在壁灯照不到的暗处消散。白色棉衣的背影被黑暗吞掉，金色头发的残留边缘在她迈入更深的暗处那一秒里闪了一下，然后彻底没了。没有回头。没有停顿。
暗街里只剩下你一个人。
右边巷道飘来的焦纸味灌进你的鼻腔，混着腐殖质的甜腻，比之前任何时候都更浓。橙色火光在巷道尽头一闪一闪的，像是在呼吸。你把她的手翻来覆去嚼了三遍——"她给了你一些东西，然后拿走了另一些东西。留给她自己的，只有亏欠"——然后你把拇指重新扣回快门边缘，握紧，松开，再握紧。相机冰冷的塑料外壳贴着你的掌心，皮带勒住后颈的重量是熟悉的，熟悉得不像是负担。
她不会再回答你了。但焦纸味没有散。

  玩家看到的正文：你站在原地没有动。焦纸味还在变浓——从右边那条稍宽的巷道里涌出来，卷着灰烬的细屑和一股灼热的炭火气，把你肺里灌满。暗街的滴水声还在身后规律地敲着金属，冷而规律。你的拇指从快门边缘移开，又扣回去——指腹贴上那个磨损的凹痕，冰凉的，熟悉的。
然后你迈步了。
不是思考之后决定的。是你的脚比你的脑子先一步做出了选择——鞋底碾碎砂砾的咯吱声连成一片，把身后壁灯漏出的斑驳光晕丢在窄巷的收拢砖墙里。右边巷道的墙壁比之前那条稍宽一些，但宽不了多少——两个人并肩走大概会蹭到彼此的肩。墙面上的青灰老砖更破了，砖缝里的黑霉蔓延成片，被不知哪来的气流撩过时掉下几粒干裂的石灰。
橙色火光在巷道尽头一闪一闪。越来越近。
你走了大概二十步——也许三十步。暗街的巷子没有参照物，时间和距离在这里都被拉长或压缩，只有焦纸味的浓度在告诉你：你正在靠近。
然后巷道弯了一下。
拐过去之后，你看到了那团火。
一个半塌的铁皮桶，侧翻在巷道尽头的石板地上，桶口涌出浓烟和忽明忽暗的橙色火焰。不是篝火——太小太集中，是有人专门在这里烧东西。桶边散着一圈灰黑色的灰烬，被巷道里穿过的气流吹出放射状的尾巴，向三四个方向铺开。火焰舔着桶口的铁皮，发出噼啪的细响，每响一次就往外溅两三点火星，落在灰堆上，熄灭，然后被新的灰盖住。
你在离桶三步的地方蹲下了。
膝盖压在冰凉的石板地上，掌心贴着地面撑住身体——不是在摆姿势，是你在战区养成的习惯：靠近一个可能有痕迹的地方，先把重心压低，降低视线，用更宽的视野扫描地面。你的相机在胸前晃了一下，皮带勒住后颈的重量把你往下拉了一寸。
灰烬还温。不是烫——是那种火焰刚熄灭没多久的余温，隔着几寸空气就能感觉到它在蒸你的脸。焦纸味在这里浓到发苦，混着烧焦的布料的刺鼻气和某种更轻的、几乎被盖过的甜腻气味——和暗街深处那股腐殖质的甜腻不一样，更干净，更淡，像是某种干花被火烧着之前的最后一丝残留。
你的手伸向灰堆边缘——然后停了。
手指在离灰烬只差半寸的位置顿住，指节僵了一瞬，然后开始轻轻颤抖。很细的颤。肉眼在昏暗里几乎不可见，但你感觉到了——指腹在空气里不受控制地弹动了两下，像是它在替你做一件你不愿做的事。你在战区见过太多被烧过的东西。纸。布。人。你知道灰烬里有时候会有什么。你的呼吸在这一秒里漏了半拍，然后你一口把这半拍吸回来，用胸腔压住，让手指继续往前伸。
指腹拨开最上面一层灰。
灰是冷的——表面的余温骗了你的脸，但灰烬内部已经凉透了。你拨开第二层。第三层。灰黑色的粉末在你指缝间滑落，露出下面没有被完全烧掉的东西：
一块指甲盖大小的布片，卡在石板缝里，边缘被烧得卷曲焦黑，但中心还残留着一点颜色——深红。不是暗红，不是锈红，是一种接近铜色的深红。和你在钟楼巷口墙根里找到的那三根卷发，颜色一模一样。
你的拇指在快门边缘停住了。
布片边缘有针脚的痕迹——不是机器的，是手工缝的。线被火烧断了，只留下几个焦黑的针孔，在灰烬的映衬下还勉强能辨认。你的指腹轻轻翻过布片，背面粘着一层极薄的灰白色粉末，在橙色火光的映照下泛着微弱的光。
暗街的滴水声从身后传过来——更远了，但还在。打在金属上，冷而规律。右边桶里的火焰又噼啪响了一声，溅出几点火星，落在你脚边，在石板地上弹了一下就灭了。
你继续拨开灰烬。手指的颤抖还在——幅度比刚才小了一点，但还没完全消失。你的另一只手无意识地从相机上滑下来，撑在地上，指节压进石板缝里的干裂泥痕，让冰凉的石面贴紧你的掌心。这个触感是你熟悉的。战场上捡拾遗物的时候，你也是这样——一只手翻找，一只手掌心贴地，用冷的、硬的、实在的东西把自己锚住。
第三层灰下面，你摸到了一片纸。
烧了三分之二。剩下的三分之一蜷曲发黄，边缘有参差不齐的黑色烧痕。你把它翻过来——纸面上有字。钢笔写的，笔画很细，但写得很用力，有几个字把纸都戳穿了，在背面凸出细小的墨点。能辨认的只剩下最后一行，其余部分已经在火焰里化成了灰，字迹被烧成一片模糊的墨渍。最后一行只残存两个半字——
“……帕……”
后面是烧焦的边。下一个字只剩一个笔画的开头，斜斜向上，然后断了。
你的手指停在那半个字上。指腹悬在烧焦的边缘，隔着不到一毫米的空气，没有按下去。你的呼吸变了——不是加速，是突然深了一拍，然后又深了一拍，像是肺忘了怎么正常换气，只能用最原始的方式把空气灌进去。你在战区见过太多的纸片。遗书。日记。被弹片撕掉一半的信。你知道这些只写了一半的笔画是什么意思——不是写不下去，是在写完之前被烧掉的。
桶里的火焰又噼啪响了一声。火星溅出来，落在那片残纸上，你下意识用指腹把它拨开——火星在你的指腹上烫出一瞬灼痛，然后灭了。
你翻遍了剩下的灰。
没有更多了。布片，残纸，几根烧焦到无法辨认的细纤维——可能是线，可能是头发，在火光下呈现出一种陈旧的金棕色，烧过的部分蜷成硬脆的小球，一碰就碎。你用指尖把它们拢在一起，拢在掌心里。你的手还在轻轻颤着，但你已经不去管它了。
灰堆旁边，石板地上有一组半截的鞋印。和你之前在岔路口发现的那些不一样——这里的鞋印更浅，更干，边缘已经在空气里风化得模糊不清，只有脚尖指向的方向还勉强可辨：往前，往桶的方向。然后转身，往巷道的另一头——暗街更深处，火光完全照不到的方向。鞋印只到灰堆就断了。另一组往深处的印子更淡，被灰烬拖出的放射状尾巴盖住大半，几乎看不出来。
暗街的滴水声还在。滴落在金属上，冷而规律。你现在离它很远——窄巷的水声已经听不太清了，取而代之的是另一种更远的滴水声，从巷道更深处传过来，打在另一块金属上，节奏和之前那个不一样，更慢，更闷。桶里的火焰在你身后把铁皮桶口烧得微微发红，发出细微的金属膨胀声，嗞——嗞——然后在下一声滴水声里沉寂。
你把布片和残纸小心地夹进相机包里——不是随便塞，是用食指和拇指捏住边缘，轻轻推进包内的夹层，不让它们再碎掉。然后你把掌心里那些烧焦的细纤维也放进去，用指腹压住，合上包盖。皮带勒住后颈的重量还在，熟悉的，冰凉的。
然后你站起来。
膝盖上的石板灰你懒得拍了。拇指重新扣上快门边缘——指腹在那个磨损的凹痕上来回划了一圈，然后停住。你看着灰堆旁那组往暗街更深处延伸的鞋印，看着它们在火光边缘模糊成一片微不可辨的阴影。橙色火光在你身后一闪一闪，把你的影子投在青灰砖墙上，拉得细长，微微抖动。
