> - **心理活动通过细微动作外化**（如抚摸猫、指尖敲桌）
> - **禁止直白抒情**

### 时间跨度的处理

**通过以下方式暗示时间**，替代"第一天""第二天"等生硬表述：
  - 光线的变化（晨曦、正午、黄昏、深夜）
  - 庆典残余的清理进度

### 7.3 悬疑的层次

- 揭示的信息让玩家感到**"知道得更多了，但问题也更多了"**
- 不一次性解释清楚所有悬念
- 保持神秘感本身就是叙事的一部分

蛰伏在下水道的老鼠们欢呼雀跃，隐忍许久的怪物们总算重见天日。
对这个日子不甚了解的人，或许会称之为米德加尔特岛的末日。
米德加尔特岛的居民更乐意称之为‘复活日’。
住在这里的所有人都很清楚要怎么活下去——要么把别人踩在脚下，要么只能被剥削。
再怎么痛苦的日子，人们也需要取乐。正值复活日，米德加尔特热闹非凡，张灯结彩的大街上充斥着欢声笑语，只有这一天，人们会稍稍放下过往的戒备，在祭典上享受自己为数不多的生命。
十一月一日，这是个好日子。
如果给法康尼一个重来的机会，他肯定会在复活日之前就和家人远走高飞，而不是继续呆在米德加尔特岛。
灯火辉煌的宴会大厅，足以容纳十数人共同进餐的长桌，却只有八人入座。大厅里不止他们四人，只是其他人都直直的站在四周，如雕像般一动不动。
上座的主人，是一名女性。
耀眼的金色长发，披在肩上的风衣和不合身的苏格兰裙实在让人不认为她有坐在上座的资格，但没人敢对她入座上座说三道四。
搓着冰冷的双手，法康尼和剩下的两人等待着上座的主人发话。
但那位主人却悠哉的握着刀叉，她轻轻割开煎得正好的牛排，送进嘴里，却皱起眉头，厌恶的砸着嘴，放下刀叉，粗鲁的用袖子擦着嘴角的酱汁。
“各位，我想你们知道今天是什么日子，这是值得开心的日子，不要摆着一副臭脸。”
略微沙哑的声音下达了‘命令’。
但是，没人能笑得出来，法康尼连头上的冷汗都不敢擦拭，目光游离在餐桌的盛宴上，不等他冷静，她又再次发话。
“昨天，我的朋友被约翰的势力袭击。万幸的是，她轻而易举的解决了来犯的敌人。我希望各位能更认真的履行自己的职责。”
她沙哑的声音仿佛带有魔力，让法康尼总算冷静下来，只是双手依旧冰冷。
那双湛蓝的眼睛扫视着在座的所有人。
沉默光临了原本应该热闹非凡的宴会。
直到，这里的主人再一次轻启朱唇。
“今天我不想追究你们的过失，谁都会有失败，这只是我们生活中的调剂品。不过，在宴会正式开始之前，我还是得作为你们的‘盟主’声明我的意见。”
这位米德加尔特的统治者高举起酒杯。
“今后，如果她食物中毒而死、不幸遭遇车祸、被雷击劈中——如果她遭遇任何意外，那我会怪罪在座的各位，请各位牢记。”
大厅里回荡着她富有磁性的声音，深深烙进在座所有人心中。
随着统治者饮下血红的酒水，复活日的狂欢正式开始。
十一月一日，这是个好日子。
今天也是卡琳娜的生日。

  "characterBeats": [{characterId, intent?, actionHint?}]（可选；角色意图或行动提示）,
  "tone": string（可选；本轮情绪基调）,
  "pacing": string（可选；给 Narrator 的呈现节奏建议，例如慢铺、快推、对话驱动、感官细节优先）,
  "requiredContent": string[]（可选；本轮必须写进正文的固定演出、固定对白、资源调用及关键顺序）,
  "currentTurnConstraints": string[]（可选；只约束本轮呈现）,
  "currentStorylineConstraints": string[]（可选；需要持续保留在当前故事线中的约束）,
  "sectionSignalSuggestion": "continue"|"maybe_complete"|"complete"（可选；当前故事线推进信号）
}

补充要求：
- involvedCharacters 必须使用稳定角色 ID。
- playerIntent 是对玩家输入性质的判断，不要把它当成剧情摘要。
- beats 只在有助于明确剧情顺序时填写。
- pacing 只写呈现方式建议，不写剧情原因。
- tone、currentTurnConstraints、currentStorylineConstraints、sectionSignalSuggestion 只在有用时填写。
- currentTurnConstraints 只约束本轮呈现；currentStorylineConstraints 用于需要在当前故事线持续保留的约束。
- 未揭示的信息不得提前暴露，除非本轮剧情明确揭示。
