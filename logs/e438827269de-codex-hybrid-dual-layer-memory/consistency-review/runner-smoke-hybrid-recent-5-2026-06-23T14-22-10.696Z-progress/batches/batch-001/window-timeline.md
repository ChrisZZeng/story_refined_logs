# Batch 001 窗口时间线

窗口范围：1-10。以下为玩家可见经历摘要，用于复盘本批次判断。

## 第1轮

- 玩家输入：无。
- 正文要点：帕兹梦见红发少女敏特在战场上死去，随后在新西西里复活日庆典上醒来，被金发少女卡琳娜踢醒。小贩介绍复活日与新西西里黑帮环境。帕兹确认自己有记者证、相机、合照和一张疑似敏特近期出现的照片。
- 选项：打听卡琳娜；追问敏特情报；质疑小贩身份。
- `preLlmEvents`：空。

## 第2轮

- 玩家输入：向小贩打听卡琳娜的事。
- 正文要点：小贩说明卡琳娜管着暗街，暗街全归她管；她不收钱，收尊严和亏欠。小贩提醒如果要找的人还在岛上，迟早要和卡琳娜打交道。
- 选项：继续追问暗街和要找的人；问小贩身份；道谢另寻线索；重新打量合照。
- `preLlmEvents`：空。

## 第3轮

- 玩家输入：继续追问暗街和你想要找的人。
- 正文要点：帕兹追问暗街位置和敏特线索，小贩明显害怕，只短暂指出暗街方向。帕兹意识到恐惧不是冲着自己，而是冲着暗街相关势力。
- 选项：直接去暗街；察觉自己被盯上。
- `preLlmEvents`：空。

## 第4轮

- 玩家输入：直接去暗街，往日总是如影随形。
- 正文要点：帕兹离开庆典走向暗街，在巷口被三个黑帮拦下。对方注意到帕兹的相机和记者身份，要求他留下相机才能走。
- 选项：交出相机；拒绝交出；示弱；往暗街深处跑。
- `preLlmEvents`：空。

## 第5轮

- 玩家输入：配合，交出相机。
- 正文要点：帕兹把相机交给为首男子，对方把相机塞进皮夹克内侧口袋，却没有让路，继续要求检查背包。三人继续缩小距离。
- 选项：拒绝交背包；交背包稳住局面；制造空隙溜进去；故意掉包争取时间。
- `preLlmEvents`：空。

## 第6轮

- 玩家输入：拒绝交出背包，挡开伸过来的手。
- 正文要点：帕兹挡开矮壮男人的手，局势升级。卡琳娜从暗处出现，制止三名黑帮并让他们退走。她解释凯旋门的人把记者往暗街钻视作间谍，并让帕兹跟她去更适合说话的地方。
- 选项：跟上卡琳娜；问凯旋门；问她为何帮忙；跟上但保持警惕。
- `preLlmEvents`：空。

## 第7轮

- 玩家输入：跟上卡琳娜，去她说的能说话的地方。
- 正文要点：卡琳娜带帕兹穿过暗街，说明凯旋门只是第一拨，之后还会有人找他。她带他到一栋老旧公寓门口，表示到了公寓或许能说些关于他要找的人和事的情报。
- 选项：跟上楼问凯旋门；先问她想得到什么；留在原地确认安全。
- `preLlmEvents`：空。

## 第8轮

- 玩家输入：跟上楼，问她关于凯旋门的事。
- 正文要点：帕兹跟卡琳娜上楼，追问凯旋门来头。卡琳娜说明凯旋门是暗街外围势力，近期因骷髅会动作变多而紧张。她提出情报以亏欠计价。
- 选项：接受交易；追问代价；观察公寓；直接说出敏特名字。
- `preLlmEvents`：空。

## 第9轮

- 玩家输入：接受交易，追问具体情报。
- 正文要点：帕兹答应欠卡琳娜一次。进入公寓后，卡琳娜说明匿名信的安排像是让他探路或当炮灰，并确认敏特在新西西里、曾在暗街出现并打听事情。卡琳娜表示敏特在找什么需要另算。
- 选项：追问敏特在找什么并接受另算；问卡琳娜为何在暗街；问猫是谁；观察公寓。
- `preLlmEvents`：空。

## 第10轮

- 玩家输入：追问敏特在找什么，接受另算的价码。
- 正文要点：卡琳娜说明敏特在找骷髅会情报，曾来找过她，之后离开。帕兹追问骷髅会是什么，卡琳娜要求更多代价，并转而询问帕兹从哪里来。
- 选项：如实回答来历；反问她为何在意；简单说明后转回交易；给出模糊回答。
- `preLlmEvents`：空。
