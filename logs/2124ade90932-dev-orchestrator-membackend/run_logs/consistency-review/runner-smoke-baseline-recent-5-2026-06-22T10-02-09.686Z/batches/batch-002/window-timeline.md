# 窗口时间线 1-20

1. 帕兹从敏特死亡的梦魇中醒来，身处新西西里复活日庆典。卡琳娜把记者证还给他，小贩提示暗街和黑帮环境。帕兹持有六人合照以及一张显示敏特可能在新西西里的匿名照片。
2. 帕兹向小贩询问敏特。小贩说暗街有消息，但要找到对的路。
3. 帕兹尝试追上卡琳娜未果。小贩进一步提示卡琳娜住在暗街，掌握很多情报。
4. 帕兹前往暗街入口，被三个凯旋门成员拦住，对方因记者和相机产生敌意并要求交出相机。
5. 帕兹假装示弱准备交出相机，卡琳娜出现制止三人，并示意这里不是说话的地方。
6. 帕兹跟随卡琳娜穿过暗街。卡琳娜解释凯旋门紧张、外乡记者危险，并把他带到自己的旧楼。
7. 帕兹进入卡琳娜公寓，观察房间、薄荷、笔记本和黑猫卡尔。卡琳娜主动说出敏特的名字。
8. 卡琳娜说敏特两周前来过，曾向她打听骷髅会情报；卡尔从沙发跳到窗台薄荷旁。
9. 帕兹说明敏特是战友且救过他的命。卡琳娜说敏特也提过战区记者，但细节有些对不上。
10. 帕兹说明收到敏特在新西西里的照片，担心她是否还活着。卡琳娜追问照片内容。
11. 卡琳娜想看那张港口区照片并询问照片用途，门外响起三声急促敲门。
12. 卡琳娜低声让帕兹别出声，同时示意他去开门；门外是陌生男子，询问卡琳娜是否在。
13. 帕兹询问男子身份。男子不答，只说找卡琳娜有事；卡琳娜在帕兹身后说自己不认识他。
14. 帕兹威胁男子离开。男子退走，卡琳娜让帕兹关门。
15. 帕兹关门后询问对方身份。卡琳娜说那人是暗街传话人，来找过两次，每次都挑她疑似独处时。
16. 卡琳娜说传话人是德索洛的跑腿，德索洛想让她查女儿出事。随后德索洛本人敲门出现，卡琳娜当面点出他收养她只是为了巴结凯旋门。
17. 帕兹斥责德索洛无仁无义，说他只在需要卡琳娜时才想起这层关系。
18. 帕兹转向卡琳娜等待她决定。卡琳娜要求德索洛称她为阁下，德索洛照做；卡琳娜同意查女儿的事，但提醒他知道她的规矩。
19. 帕兹继续保持沉默。正文重新演出德索洛称呼阁下、卡琳娜同意调查和规矩提醒，并额外写成德索洛单膝跪地、握住卡琳娜指尖。随后卡琳娜让他明天再来，并说明暗街规矩。
20. 帕兹跟进卡琳娜回公寓。卡琳娜说帕兹刚才一句话都没说，问他是否觉得德索洛可怜。
