你站在门内的第一级台阶上。
身后传来了门轴转动的声响——不是很大的声音——是卡琳娜在你落脚之后，伸手拉上了门板。门框与门板之间的接触面贴合时发出一声钝响，然后室外路灯的光线被切断，视野中的亮度瞬间下降了一个等级。
黑暗。
不是完全的黑暗——眼睛在大约两次呼吸的时间后开始适应。通道中有光，不是来自室外，是来自更下方的一个微弱的光源——距离很远，亮度极低，像是隔了几层纱在看一根蜡烛。它在地面和墙壁上投下一种不确定的、浮动性的微光，刚好足够让眼睛辨认出石阶的轮廓和两侧墙壁的边界，但不足以看清细节。
卡琳娜在你面前迈开了步子。
她能看见。你从她下脚的方式中确认了这一点——她的步子没有试探，没有犹豫，每一步都落在石阶的中间位置，鞋底与石面接触时发出的声响均匀而稳定。她抱着卡尔向下走去，黑猫在她臂弯中没有动——尾巴垂落的姿势也没有变化。
你跟上她。
第一级台阶到第二级之间的距离比普通楼梯略高——不是一个标准的建筑尺度，是手工凿出来的台阶，每级的高度和深度都有细微的差异。你需要略微调整自己的步幅来适应这个节奏。石阶的表面是粗糙的——鞋底落在上面时能感觉到石面不规则的纹理，有些地方有轻微的凹陷，像是被无数只脚踩过后磨损的痕迹。
通道向下延伸。
不是笔直的——它在向下大约六七级台阶后向右侧拐了一个平缓的弯，然后继续下行。拐弯处的墙壁上有一道细长的裂缝，从墙顶延伸到接近地面，裂缝边缘呈深色，像是长期有水分从那里渗出来留下的痕迹。
空气在变化。
温度在每一级台阶的下降中降低了一个几乎不可察觉但确实存在的梯度——从外面夜间的凉意，过渡到一种更恒定的、属于地下的凉。不是阴冷，是一种封闭空间特有的那种均匀的温度。空气中混合着石头的气味——不是潮湿的，是干燥的石灰岩在长期与空气接触后释放的那种矿物质气息，带一点微弱的铁味。还有泥土——压实了的、被时间和脚步磨成了细粉的泥土，在你们经过时从地面上扬起极细微的颗粒，进入鼻腔。
你们继续下行。
脚步声在通道中产生了另一种质地的回响——不是石阶本身的声音，是通道的截面宽度和高度形成的天然共鸣腔，对特定频率的声音产生了放大效果。鞋底落地的声响被吸收了一部分，被反射了一部分，在你的听觉中形成一种与室外完全不同的空间感——像是在一个空的大房间里说话，即使音量不高，声音也会在周围停留片刻。
卡尔在卡琳娜臂弯中睁开了眼睛。
不是很快的动作。它的眼睑在你们拐过那个弯道后缓慢抬起——先露出瞳孔的上缘，然后瞳孔完全暴露在微光中。它的瞳孔在适应这个光线水平时发生了一个快速的变化——从较大直径收缩到中等，然后根据光量重新调整到一个稳定的尺寸。
它的耳朵朝后转动了一下——不是朝你的方向，是朝通道前方的某个点，像是它正在用耳朵测量那里距离入口有多远。然后它的耳朵重新朝向侧方。
它的尾巴从垂落的状态松开——不是紧张地竖起来，是一种从盘绕或收拢的状态中舒展开来，沿着卡琳娜的前臂外侧垂落，末端在空气中画了一道缓慢的弧形，然后静止。
卡琳娜没有说话。她的步子保持着那个节奏，向下，向下。
通道的光线在继续变化——不是变亮，是光源的距离在缩短。你能感觉到那个光在你们前方某个位置等待，但它的亮度增长得非常缓慢，像是被距离磨钝了。
大约又走了十几级台阶。
通道在这里结束了——不是突然结束，是台阶在最后一级处变成了一个平坦的、大约三步宽的过渡空间，然后前方出现了一道门。
一扇旧木门。
半掩着。
它的门板由几块纵向拼接的厚木板组成，木板之间的缝隙被岁月和灰尘填满，形成一道道深色的线条。木料的颜色在微光中呈现出一种深褐色的调子——不是漆面的颜色，是木材本身在长期空气接触中氧化后形成的自然的深色。门框与墙壁之间的连接处能看到铁质的合页——不是铁匠铺里打的新东西，是老旧的、被时间和水分侵蚀后表面带着一层深色氧化层的铁。
门缝里透出光。
不是明亮的——是一种暖黄色的微光，像是从一个低瓦数的灯泡或一盏油灯中发出来的。光从门缝中溢出，在地面上投下一个狭窄、扁平的亮带，边缘柔和，与通道地面的暗色形成温和的对比。光在亮带中呈现出一种缓慢的、几乎不可见的脉动——不是电灯的稳定，是一种更接近火焰的光特有的那种不规则的、有生命感的亮度变化。
卡琳娜站在那扇门前。
她没有敲门。她伸出手——右手从卡尔的背部移开，落在门板上。指腹与木料接触的位置大约在门板中间偏左的地方。她推了一下。
门轴没有发出声音。
门向内侧打开了。
光从内部空间中倾泻出来——不是突然的，是随着门板的转动，光逐渐填充了门框的范围，在你的瞳孔中完成了一次缓慢的、温和的亮度适应切换。
你看到了那个空间。
不是一个很大的空间——大约普通房间的大小。天花板不高，由原木横梁支撑，梁木表面的颜色已经深到接近黑色，布满了细密的纵向裂纹。墙壁是石灰粉刷的——不是平整的现代抹灰，是手工涂抹的，表面带着不均匀的纹理和细微的起伏。地面上铺着老旧的木地板——木板宽度不一，边缘磨损，踩上去会有一种略带弹性的感觉。
空间中的光源来自壁炉——一个老旧的石砌壁炉，炉膛里正燃着一小堆火。火焰不大，刚好足够在空间中制造出一个温暖的光区，并与室外的凉意形成对比。壁炉台上放着几样东西——一个陶罐，一根熄灭的蜡烛，一块被火焰烘烤后边缘卷曲的松木块。墙壁上的阴影在火光中缓慢移动。
空间里还有一张桌子——木质的，表面有磨损的痕迹，桌面上放着一个杯子，一个碟子，一块叠好的深色布料。一把椅子靠在墙角——不是匹配的家具，是一把单独存在的、看起来是从别处搬来的木椅。墙角堆着几本书——书脊上的字在火光中看不清楚，只能辨认出那是书。
空间不大，但它是完整的。是有温度的。是有人住的样子。
卡琳娜站在门内侧。
她侧过了身——不是完全转向你，是一种半侧的姿态，身体的一部分仍然朝向着内部空间的方向，但肩膀和目光都向你敞开了一个角度。她的右手仍然搭在门板上，左手抱着卡尔。黑猫在她臂弯中睁着眼睛，瞳孔在火光中收缩成细窄的椭圆形，目光在空间中缓缓扫过一遍——确认一切如常——然后重新闭上了眼睛。
卡琳娜开口了。
她的声音在这个空间中听起来与室外不同——不是音量不同，是音质发生了变化。通道和石墙的声学环境让她的声音多了一层自然的共鸣，少了一层空气中传播时的衰减感。她的语调也比户外时更放松——不是松懈，是句子之间的间隔更自然了，像一个人在自己家里的光线和温度中不需要再做额外的调整。
[卡琳娜]“进来吧。”
她说这个词时，没有用她在外面时那种简洁的、带着距离感的语调——是更接近之前在长椅上说话时的声音底色。她在说完后，从门边向后退了半步——让出了入口的宽度。她抱着卡尔站在桌子旁，火光在她侧脸上投下一个从暖黄到暗影的渐变，她看着你，等待着。
暖光从她身后流出来，落在门槛前的地面上，形成一个欢迎的、温和的亮区。
</turn>

<output_requirements>
Director 本轮只输出结构化 JSON。

<output_schema>
只返回一个 JSON 对象，不写正文、对白、旁白或解释。

字段结构如下，字段名保持英文：
{
  “playerIntent”: string,
  “summary”: string,
  “involvedCharacters”: string[],

  “beats”: string[],

  “pacing”: string,
  “requiredContent”: string[],

可选字段没有内容时省略，不输出 null。

<field_guide>
playerIntent（可选）——对玩家输入性质的判断：行动、探索、表达、对抗、初始开场。这是分类，不是剧情摘要。

summary——一句话概括本轮发生的剧情。供系统和 Narrator 快速把握全局。

involvedCharacters——本轮行动、说话，或需要被正文感知到在场/产生影响的角色 ID，使用稳定 ID。玩家主角若是本轮行动或观察的承载者，也可以作为参与角色列入，帮助 Narrator 把动作落在具体人物上。所有需要被正文感知到的角色，都应在 beats 或 characterBeats 中有可见位置。

scene（可选）——场景或位置发生变化时填写。

beats——本轮正文的剧情骨架，按顺序列出关键节拍。第一条 beat 是本轮正文的起点，最后一条是终点——Narrator 从第一条开始写，写完最后一条后停下。写事件和动作，不写文学描写。
  ✅ “帕兹向卡尔追问铃铛来历 → 卡尔用反问引导帕兹面对记忆空白 → 帕兹承认记忆分裂”
  ❌ “月光洒落石板路，脚步声在空旷巷子里回荡”

characterBeats——基于 beats 为重要角色提炼角色执行骨架：角色在本轮节拍中想达成什么、以怎样的动作、反应或语气推进。它帮助 Narrator 把 beats 落成具体角色表现，并补足角色层面的动作质感、潜台词和信息边界。
  - intent：角色本轮的意图或目标
  - actionHint：角色具体做什么，要具体到 Narrator 可执行。同时作为潜台词层——将隐藏信息翻译为可观察的行为质感，告诉 Narrator”怎么表现这个角色”而不是”角色背后是什么设定”
  ✅ actionHint: “沉默三秒后转移话题，语气刻意轻松”
  ❌ actionHint: “做出反应” / “她其实是眼线所以在套话”

tone（可选）——本轮情绪基调。给出明确方向。
  ✅ “表面平静但暗流涌动，对话中有试探和隐瞒”
  ❌ “适当的氛围”

pacing（可选）——给 Narrator 的呈现节奏建议。只写呈现方式，不写剧情原因。

requiredContent（可选）——本轮必须出现在正文中的固定演出：剧本设定的台词、固定剧情表演、引擎资源调用。
如果固定演出或固定对白有剧本顺序，必须先按顺序写入 beats 的对应位置；requiredContent 只记录不可漏、不可改义的具体内容，不单独承担顺序控制。

currentTurnConstraints（可选）——本轮 Narrator 需要遵守的约束集合。可以包含本轮新增的临时限制，也可以从 turn_constraints 中挑出本轮特别需要强调的条目。这是 Narrator 唯一能看到的约束来源。

currentStorylineConstraints（可选）——本轮新产生的、需要在后续持续遵守的约束。

sectionSignalSuggestion（可选）——当前故事线推进信号。continue：仍在推进中；maybe_complete：接近完成，本轮应有收束感；complete：本节点已完成。
</field_guide>

关于 requiredContent：
- 只放本轮必须出现在正文中的固定演出内容：剧本设定的台词、固定剧情表演、引擎资源调用（backgrounds、characters、cgs 等）
- 普通约束不往这里放

- beats 写清本轮事件顺序和推进边界；characterBeats 从这些事件中提炼重要角色的执行方式
- 主角行动若承载本轮重点，把主角也作为重要角色处理，写清他的行动意图、观察方式或身体反应。

- plot_content 中已经给出先后顺序的固定演出、固定对白，必须按原顺序消费，不得提前、延后、合并或跳过

- 这是 Narrator 唯一能看到的约束来源，你必须把本轮 Narrator 需要遵守的约束全部放在这里
- 从 turn_constraints 中选取本轮仍然有效且需要强调的条目，加上本轮新增的临时限制
